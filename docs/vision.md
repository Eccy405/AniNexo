# Visión del Proyecto: AniNexo

## Definición
AniNexo es una plataforma híbrida diseñada para ser el punto de encuentro definitivo de la comunidad de anime. No es simplemente una red social, ni simplemente una base de datos; es una **Wikipedia de Anime con una Red Social Integrada**.

## Pilares Fundamentales

### 1. La Enciclopedia (Wikipedia)
*   **Fuente de Verdad:** Integración profunda con la API de AniList para obtener datos precisos de series, películas, personajes y staff.
*   **Capa de Conocimiento:** Un sistema de caché inteligente (Redis/MySQL) que permite una navegación rápida y offline-first para datos consultados frecuentemente.
*   **Fichas Técnicas:** Páginas detalladas que no solo muestran información, sino que sirven como el "ancla" para toda la actividad social.

### 2. Ecosistema Social Integrado
*   **Socialización Contextual:** Los usuarios pueden interactuar directamente sobre el contenido de la enciclopedia (ej. comentar en un episodio específico, dar 'like' a un personaje).
*   **Feed Dinámico:** Un muro social que combina actualizaciones de amigos, noticias de la enciclopedia y recomendaciones de IA.
*   **Mensajería Real-Time:** Chat 1a1 y grupal integrado para discutir teorías, episodios o simplemente socializar.

### 3. Nexo (Inteligencia Artificial)
*   **El Bibliotecario Inteligente:** Una IA que no solo chatea, sino que "entiende" la enciclopedia. Puede recomendar animes basados en gustos complejos, explicar lore y ayudar en la moderación.
*   **Diferenciación de Valor:** Funciones de IA avanzadas para usuarios Premium, proporcionando un análisis más profundo y una interfaz más rica.

## Evolución
Este documento se actualizará conforme el proyecto evolucione para asegurar que el "Norte" técnico y de producto siempre esté claro y documentado.
